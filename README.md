# smallzh frontend-dev

记录对于浏览器页面开发相关的ES6/TypeScript/React18+/Tailwind CSS的相关知识点

## 1.安装依赖

```shell
# 安装依赖
uv sync
```

## 2.启动
```shell
uv run mkdocs serve
```

## 3.运行demo

### 1 React18 开发服务器
npm run dev:react18
### 2 Tailwind CSS 开发服务器
npm run dev:tailwindcss
### 3 TypeScript 测试
npm run test:typescript
### 4 ES6 测试
npm run test:es6
